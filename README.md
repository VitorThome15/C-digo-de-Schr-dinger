## 📝 Sobre o projeto
Este software tem como objetivo auxiliar a instituição **SANEM** na organização, gerenciamento e registro de suas doações, garantindo maior controle e transparência nos processos de arrecadação e distribuição.

### 🚀 Funcionalidades Principais
* Registro detalhado de doadores e donatários.
* Controle de inventário de itens físicos.
* Gestão de causas e metas de arrecadação.
* Monitoramento de fluxos financeiros e materiais.
